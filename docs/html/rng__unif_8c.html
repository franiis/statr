double unif_rand ( RNG_state_t **  state)

Generates pseudorandom number based on Mersenne-Twister algorithm from uniform distribution.

Parameters
stateobject storing state of RNG. Passing NULL will create random seed based on time. Frequent calling this method while passing NULL can generate many same numbers (based on sam seed). First call with given object or with NULL can take longer due to RNG state initialization. Also regeneration of RNG state is required after each 624 numbers.
Returns
Random number from uniform distribution based on passed RNG state.
Warning
For generation numbers from uniform distribution use Instead use double runif(RNG_state_t **state, double a, double b)
